# Análisis de Comportamiento de Consumo — San Valentín
### Herramienta: Power BI

## Descripción
Dashboard interactivo desarrollado en Power BI para analizar patrones 
de gasto y comportamiento de consumidores en San Valentín. Incluye 
segmentación por género, estado civil y tipo de regalo, con filtro 
dinámico por rango etario.

## Hallazgos principales
- Las mujeres representan el 60,75% del gasto total
- Las flores son el regalo más elegido con un 28,93% de preferencia
- El grupo de 25-34 años es el de mayor gasto

## Habilidades aplicadas
Power BI · Power Query · Modelado de datos · DAX · Visualización
